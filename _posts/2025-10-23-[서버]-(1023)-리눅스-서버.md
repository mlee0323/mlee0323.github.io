설치한 패키지

- net-tools, mysql-server, vsftpd, php, apache2, libapache2-mod-php, php-mysql

sudo systemctl enable mysql

- 부팅시 자동시작

mysql 설치 후 루트 PW바꿔주고  필요시 새로운 계정 생성

- sudo mysql -u root
- alter user ‘root’@’localhost’ identified with mysql_native_password by ‘’;
- flush privileges;

임시 ip 추가


sudo ifconfig ens33:0 <ip> up

- down 하면 ip 삭제
- 재부팅해도 삭제

/ete/apache2/sites-available 에 000-default.conf 있는데 안에 virtualhost 관련 설정 존재


우리가 한건 임시 ip 추가 후 docroot 을 각각 ip에 맞게 새로 부여


여기서는 /var/www/www, /var/www/msp4, /var/www/msp5

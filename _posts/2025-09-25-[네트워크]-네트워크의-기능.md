계층 모델 : OSI 7계층(layer)


TCP/IP 4계층

- 응용계층 - HTTP
	- 인터넷을 구성하는 각기 다른 물리적 네트워크의 연결과 관련된 기능을 수행
	- 응용, 표현, 세션
- 전송 계층 - TCP, UDP
	- 데이터를 패킷으로 분할하고 목적지까지 전송 경로를 설정
	- 전송 계층
- 인터넷 계층 - IP
	- 송수신 사이에서 데이터를 주고받는것
	- 네트워크 계층
- 네트워크 접근 계층 - 이더넷
	- 데이터 링크 계층, 물리 계층

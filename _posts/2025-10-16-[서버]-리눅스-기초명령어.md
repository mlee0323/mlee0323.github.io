| 명령어   | 기능                                                                                     |
| ----- | -------------------------------------------------------------------------------------- |
| ls    | ls : 기본 폴더 내 해당되는 파일 조회
ls -l : 상세내용 조회
ls -a : 숨김파일 포함 조회                             |
| rm    | rm a.txt : 파일 삭제
rm -rf : 폴더 및 파일 삭제                                                   |
| cat   | cat a.txt : 해당 파일 내용 조회                                                                |
| cd    | cd home : home 디렉토리로 이동                                                                |
| mv    | mv a.txt home : 파일을 이동                                                                 |
| cp    | cp a.txt b.txt : a파일을 b로 변경후 복사
cp a.txt home : a파일을 home 폴더로 복사. home/b.txt 하면 이름도 변경 |
| mkdir | mkdir house : 새로 house 폴더 생성
mkdir -p house/my/bed : -p 사용시 house/my 디렉토리가 없으면 같이 만듬   |
| pwd   | 현재 폴더 위치 표시                                                                            |
| touch | 그냥 수정일시만 변경. 그 외 기능 x                                                                  |
| more  | more a.txt : 창 크기만큼 파일 내용 보여줌, 내용이 더 많으면 넘어감
more +10 a.txt : 10줄씩 표시                  |
| find  |                                                                                        |
